To connect your Charles Schwab brokerage account, you need a Schwab Developer API app. This is free and takes about 5 minutes.

  1. Register at developer.schwab.com
    Create a free account and add a new app.
  2. Set your callback URL
    In your app settings, set the callback URL to:
    {{ request.host_url }}schwab/callback
  3. Copy your credentials
    Copy the App Key (client ID) and App Secret.
  4. Set environment variables
    SCHWAB_CLIENT_ID=<your App Key>
SCHWAB_CLIENT_SECRET=<your App Secret>
SCHWAB_REDIRECT_URI={{ request.host_url }}schwab/callback
    On Render/Railway: set these in your service Environment settings.
    Local dev: add to a .env file (never commit to git).
  5. Restart the app, then return here and click Connect.
Security: Credentials are stored as environment variables only. Tokens are stored in the app database and never exposed in the UI. This integration is read-only — no trades can be placed.
